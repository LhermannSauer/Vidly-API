Vidly API

Welcome to the online API for the Vidly Application




This is a project done as part of the Node JS course by CodeWithMosh. It serves as a backend using mongoDB for a react project called Vidly, which will be also uploaded shortly.

The course comprehensively teaches you how to write a clean and consisten node application, covering from the basics concepts such as the module system, the Node Package Manager, Express, Asynchronous JS, CRUD operations using Mongoose; up to advanced topics such as authorization and authentication, unit and integration testing, Test-Driven-Development, and deployment.
